# ottbike_counters
Ottawa Bike counters: historical data and forecasting with R

Shiny app featuring:
-  time series plots
-  seasonal plots (annual, hebdo)
-  bike counts by weather conditions
-  autocorrelation
-  PCA by time series features
-  intercorrelation scatterplots
-  map of the area
